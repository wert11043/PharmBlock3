# Pharm Block 3 Quiz

以藥理 Block 3 總複習表為基礎製作的靜態配合題網站，可直接部署到 GitHub Pages。

## 內容

- `index.html`：網站主頁
- `styles.css`：版面樣式
- `app.js`：配對互動邏輯
- `quiz-data.js`：題庫資料
- `.nojekyll`：讓 GitHub Pages 直接用靜態檔部署

## 練習模式

- 藥物群組：`抗血小板`、`抗凝血/溶栓/止血`、`造血系統`、`全部混合`
- 練習欄位：`機轉/標的`、`適應症`、`副作用 / 禁忌`、`解毒 / 逆轉`、`監測 / 注意事項`、`考點速記`

## GitHub Pages

若此 repo 啟用 GitHub Pages，預期網址會是：

`https://wert11043.github.io/PharmBlock3/`

建議在 GitHub repo 設定中選擇：

- `Settings`
- `Pages`
- `Build and deployment`
- `Source: Deploy from a branch`
- `Branch: main`
- `Folder: / (root)`
